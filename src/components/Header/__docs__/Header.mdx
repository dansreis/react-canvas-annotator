# Header

Header component with different props.

#### Example

<Canvas of={Header.Full} />

## Usage

```ts
import {Header} from "react-image-annotator";

const Example = () => {
  return (
      <Header
        primary
        size={"medium"}
        imageInfo={{width: 1920, height: 1080}}
        zoom={{value: 10, action: (event) => console.log(event)}}
        annotation={{value: true, action: (event) => console.log(event)}}
      />
  );
};

export default Example;
```

#### Arguments

- **text** _`() => void`_ - A string that represents the text content of the button.
- **primary** - A boolean indicating whether the button should have a primary styling or not. Typically, a primary button stands out as the main action in a user interface.
- **size** - A string with one of three possible values: "small," "medium," or "large." It defines the size or dimensions of the button.
- **imageInfo** - Image width and height.
- **zoom** - An object with the "value" of current zoom and an "action" to be performed on click.
- **annotation** - An object with the "value" of enabled/disabled annotations, as well as an "action" to be performed on click.
