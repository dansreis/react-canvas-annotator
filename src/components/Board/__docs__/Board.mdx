# Board

Button component with different props.

#### Example

<Canvas of={Board.Main} />

## Usage

```ts
import Board, {BoardActions} from "react-canvas-annotator";

const Example = () => {
  const image = {name: "test", src: "http://imagesource.com/test.png"}
  const ref = React.createRef<BoardActions>();
  return (
      <Board
        ref={ref}
        image={image}
        items={[]}
      />
  );
};

export default Example;
```

#### Arguments

- **image** `{ name: string, src: string }` - The image object containing the name and source URL.
- **items** `CanvasObject[]`: - An array of canvas objects to be rendered.
- **initialStatus** `{ draggingEnabled?: boolean, currentZoom?: number, scaleRatio?: number }`: - Optional initial status settings for the board:

`draggingEnabled`: Boolean flag indicating if dragging is enabled.

`currentZoom`: Number representing the initial zoom level.

`scaleRatio`: Number representing the scale ratio.

`onResetZoom () => void`: Optional callback function to reset the zoom level.

`onZoomChange: `(currentZoom: number) => void` - Optional callback function called when the zoom level changes.

`currentZoom`: Number representing the current zoom level.

- **currentStatus**: Boolean indicating the current dragging status.
- **onLoadedImage**: `({ width, height }: { width: number, height: number }) => void` -  Optional callback function called when the image is loaded (width: Width of the loaded image, height: Height of the loaded image).
